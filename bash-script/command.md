# Bash Script

### Example: Simple Bash Script
`script.sh`
```bash
#!/bin/bash
echo "Hello, Sourov!"
```
`#!/bin/bash` `#!  → shebang` এর কাজ হলো এই স্ক্রিপ্টটা কোন প্রোগ্রাম দিয়ে চালাতে হবে সেটা অপারেটিং সিস্টেমকে জানানো।

### Example: Using Variables
`script.sh`
```bash
name="Sourov"
echo "Hello, $name!"
```
